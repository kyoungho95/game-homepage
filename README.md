1. 프로젝트 개요 
★ 프로젝트 개요 : 1-1 컨셉
- 게임 시작을 시작할 수 있는 홈페이지를 구현
- 홈페이지 내 유저들에게 이벤트, 공지사항 등 정보 제공
- 유저들간 소통을 할 수 있는 커뮤니티 제공
- 포인트 상점 기능을 구현 하여 게임활동 및 커뮤니티 활성화
- 홈페이지 이용 및 불만사항 개선을 위한 고객센터 구현
